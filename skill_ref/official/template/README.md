# template - 基础模板

## 来源信息
- **原始来源**: https://github.com/anthropics/skills/blob/main/template/SKILL.md
- **Stars**: 109k (anthropics/skills 仓库)

## 亮点说明

### 1. 最简 Skill 模板
- 最小化的 frontmatter 结构
- 清晰的必填字段：`name` 和 `description`
- 简洁的指令区域

### 2. 标准 frontmatter 格式
```yaml
---
name: template-skill
description: Replace with description of the skill and when Claude should use it.
---
```

### 3. 清晰的分区
- Frontmatter 元数据区
- 指令内容区
- 简洁的页脚

### 4. 可扩展性
- 作为创建新 Skill 的起点
- 鼓励根据需求添加更多 section
